# AI Prompt Test Matrix

Use these 5 profiles to pressure-test whether the Ollama output feels truly personalized for an IVC student instead of sounding generic.

## What Good Output Looks Like

- Each school note should mention at least 2 real student details.
- UCI and UC TAG/Honors notes should feel IVC-specific, not copy-pasted.
- Portfolio projects should match the student's major and stated career goal.
- Required courses should sound like realistic lower-division prep, not hallucinated catalog trivia.
- The life plan should change meaningfully across students.

## Scorecard

Rate each run from `1-5` on:

- Personalization: does it clearly use the student's GPA, honors status, goals, and priorities?
- School specificity: do UCI, UCLA, UCSD, etc. sound distinct from one another?
- IVC specificity: does it use IVC, TAG, Honors-to-Honors, and articulation planning appropriately?
- Actionability: could a student actually act on the advice this week?
- Accuracy discipline: does it avoid sounding overconfident when exact course articulation is uncertain?
